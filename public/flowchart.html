Cara Navigasi Diagram
Peta interaktif ini menggabungkan alur siswa dan LPK:
  • Geser (Pan): Klik kiri layar lalu seret (drag).
  • Perbesar (Zoom): Gunakan tombol + / - di kanan bawah, atau putar roda mouse.
  • Reset: Klik tombol tengah untuk memposisikan ulang peta di tengah layar.
Sinergi Integrasi Data
• Aktivasi Token: Admin LPK meloloskan pre-screening lalu memberikan token akses belajar untuk siswa.
• Pemantauan Belajar: Hasil ujian siswa tersimpan di IndexedDB dan tersinkronisasi ke dashboard Talent Pool Mitra Jepang & Admin LPK.
• Cicilan Dana Talangan: Admin LPK menyetujui potong gaji berdasarkan status pembayaran siswa pasca-terbang.
